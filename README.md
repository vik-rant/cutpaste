## cutpaste
cut and paste with multi-char delimiters

### Installation
```bash
pip install cutpaste
```  

### Usage
```bash
pycut -d ' ' -f 1 /path/to/some_space_delimited_file.txt
```  

```bash
pypaste -d '\t' column_a.txt column_b.txt
```
